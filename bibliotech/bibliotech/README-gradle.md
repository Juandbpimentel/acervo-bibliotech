Este módulo foi convertido de Maven para Gradle.

Como usar:

- Usando o Gradle Wrapper (recomendado):
  - Windows (cmd): `gradlew.bat clean build`
  - Unix: `./gradlew clean build`

- Usando Gradle instalado globalmente:
  - `gradle clean build`

Observações:
- O `pom.xml` original foi preservado para histórico; você pode removê-lo se não precisar mais.
- Para gerar a wrapper novamente com uma versão específica do Gradle: `gradle wrapper --gradle-version <versao>`
