# Projeto Acervo Bibliotech - Engenharia de Software I

Este repositório contém o código-fonte do projeto "Acervo Bibliotech", desenvolvido como parte da disciplina de "Projeto Integrado em Engenharia de Software I". O objetivo foi criar um sistema de gerenciamento de acervo para uma biblioteca, aplicando na prática todo o ciclo de vida do desenvolvimento de software.

Este projeto foi uma oportunidade única de vivenciar os desafios e as recompensas de cada etapa da engenharia de software, desde a concepção até a entrega, e serviu como uma base sólida para o desenvolvimento de competências técnicas e gerenciais.

## Visão Geral do Projeto

O "Acervo Bibliotech" é uma aplicação desktop (via Electron) com um backend robusto que visa modernizar e otimizar a gestão de uma biblioteca. O sistema oferece funcionalidades como cadastro de livros, gerenciamento de empréstimos, controle de usuários e organização do acervo em estantes e seções.

*   **Frontend:** Desenvolvido em **React com TypeScript**, proporcionando uma interface de usuário moderna, reativa e de fácil utilização.
*   **Backend:** Construído em **Java com Spring Boot**, garantindo uma API RESTful segura, escalável e de alta performance.
*   **Banco de Dados:** Utiliza **PostgreSQL**, um sistema de gerenciamento de banco de dados relacional poderoso e confiável.
*   **Containerização:** O ambiente de desenvolvimento e produção é gerenciado com **Docker**, facilitando a configuração e a implantação.

## Competências Desenvolvidas

Esta experiência acadêmica foi fundamental para consolidar conhecimentos práticos alinhados às demandas do mercado, especialmente para a vaga de desenvolvedor júnior. As atividades realizadas me permitiram:

### Engenharia de Software e Gestão
*   **Levantamento e Análise de Requisitos:** Conduzi entrevistas com stakeholders para definir o escopo e os requisitos funcionais e não funcionais do sistema.
*   **Modelagem e Arquitetura:** Participei ativamente da criação de diagramas (UML, ER) e da definição da arquitetura do software, escolhendo as tecnologias e os padrões de projeto mais adequados.
*   **Gerenciamento de Projeto:** Colaborei na organização de tarefas, no planejamento de sprints e na utilização de um cronograma para garantir as entregas, simulando um ambiente de desenvolvimento ágil.
*   **Documentação Técnica:** Ajudei a criar e manter a documentação do projeto, incluindo manuais de usuário e especificações técnicas, uma habilidade crucial para o suporte e a manutenção de sistemas.

### Desenvolvimento e Tecnologia
*   **Criação e Manutenção de Aplicações Web:** Atuei no desenvolvimento full-stack, desde a criação de componentes de interface com **React e TypeScript** até a implementação de regras de negócio e endpoints da API com **Java e Spring Boot**.
*   **Frontend (HTML/CSS/JavaScript):** Desenvolvi interfaces responsivas e interativas, utilizando **HTML5 semântico**, **CSS3** (com Bootstrap) e **JavaScript (ES6+)** para criar uma experiência de usuário fluida.
*   **Backend e APIs:** Implementei a lógica de negócio, a camada de acesso a dados (JPA/Hibernate) e a segurança da aplicação no backend, expondo os serviços através de uma **API RESTful**.
*   **Bancos de Dados (PostgreSQL):** Modelei o banco de dados relacional e escrevi consultas SQL para manipulação dos dados, compreendendo a importância da integridade e da performance.
*   **Controle de Versão com Git:** Utilizei o **Git** para gerenciar o código-fonte, trabalhando com branches, commits semânticos e pull requests, seguindo as melhores práticas de desenvolvimento colaborativo.
*   **Noções de Docker:** Contribuí para a criação e manutenção dos `Dockerfile` e `docker-compose.yml`, entendendo como containerizar a aplicação para garantir um ambiente de desenvolvimento e implantação consistente.

Este projeto demonstra minha capacidade de aprender rapidamente, de trabalhar em equipe e de aplicar conhecimentos teóricos para resolver problemas práticos, desenvolvendo uma aplicação completa e funcional do zero.
